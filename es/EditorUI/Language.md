# Switch application language

You can switch between these languages:

* <u>English (Default)</u>
* <u>Chinese (Simplified)</u>
* <u>Dutch</u>
* <u>French</u>
* <u>German</u>
* <u>Italian</u>
* <u>Polish</u>
* <u>Portuguese (Brazilian)</u>
* <u>Portuguese (European)</u>
* <u>Russian</u>
* <u>Spanish</u>
* <u>Ukrainian</u>
* etc.

To switch a language go to the `Help -> Language -> [the language]` menu.
